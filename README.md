# expressjs form

Form that validates data and stores it in MongoDB.

## Specifikace

- Frontend: ReactJS, TypeScript
- Backend: ExpressJS, TypeScript, Mongoose
- Database: MongoDB

## How to run:

- Frontend:
  ```
  cd client
  npm i
  npm start
  ```
- Backend:

  .env file with the following values is needed: ```DATABASE_URL```, ```PORT```.
  ```
  cd server
  npm i
  npm run dev
  ```
