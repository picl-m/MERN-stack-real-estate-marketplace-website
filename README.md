# reas-projekt

Formulář s několika kroky a validací údajů.

## Specifikace

- Frontend: ReactJS, TypeScript
- Backend: ExpressJS, TypeScript, Mongoose
- Databáze: MongoDB

## Jak projekt spustit:

- Frontend:
  ```
  cd client
  npm i
  npm start
  ```
- Backend:

  Je potřeba vytvořit .env soubor s hodnotami: ```DATABASE_URL``` a ```PORT```.
  ```
  cd server
  npm i
  npm run dev
  ```
